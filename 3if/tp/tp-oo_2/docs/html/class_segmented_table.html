Detailed Description

template<typename E>
class SegmentedTable< E >

A table which can adjust its size in large chunks and works as an array.

The table is made up of multiple segments of a preset size. When a segment runs out of size to store data, a new segment is allocated. Accessing elements works like an array, since the segments are stored in order.
